---
title: Font Regular
---

By activating this option, the `font-normal` class will be renamed to `font-regular`.

```css
@theme {
    --font-weight-normal: initial;
    --font-regular: 400;
}
```

## Import

This utility is NOT imported by default.

```css
@import "tailwindcss-addons/theme/font-regular.css";
```
