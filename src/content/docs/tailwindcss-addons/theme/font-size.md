---
title: Font Size
---

```css
@theme {
    --text-2xs: 0.625rem;
    --text-2xs--line-height: calc(1 / 0.625);

    --text-3xs: 0.5rem;
    --text-3xs--line-height: calc(1 / 0.5);
}
```

## Import

This theme is imported by default.

### Manual

```css
@import "tailwindcss-addons/theme";
/* Or */
@import "tailwindcss-addons/theme/font-size.css";
```
